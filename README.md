# OpenLinkage

OpenLinkage 是一个开源、浏览器端的平面机构设计与自动综合平台。

项目将从四杆机构实验室开始，逐步支持目标轨迹拟合、六杆腿、可变几何机构和绳驱动手指。

## 本地开发

```bash
npm install
npm run dev
```

打开 `http://localhost:3000`。

## 路线图

- 0.1：四杆机构编辑、动画、轨迹和 Grashof 判断
- 0.2：通用平面约束机构
- 0.3：手绘轨迹与四杆自动拟合
- 0.4：Klann / Stephenson 腿部模板
- 0.5：可变几何机构的多轨迹综合
